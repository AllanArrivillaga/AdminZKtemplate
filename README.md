# How to Run
## No Maven installed
You can run the project with maven wrapper by the command below (it will automatically download required maven):

### Linux/Mac
`./mvnw jetty:run`
### Windows
`mvnw jetty:run`

## Maven installed
With the command below:

`mvn jetty:run`

After the jetty started
visit [http://localhost:8080](http://localhost:8080)
